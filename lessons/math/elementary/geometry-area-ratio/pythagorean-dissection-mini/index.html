勾股定理（最小试点）

目标：快速验证“3Blue1Brown 风格可视化 + 交互课件”混合路线可行性。

已知直角三角形两条直角边分别为 3 和 4，斜边为 5。用“面积重排”直观看到为什么 3² + 4² = 5²。
方法A（重排证明，通用）
构造边长为 (a+b) 的大正方形，放入 4 个全等直角三角形（直角边 a、b，斜边 c），中间会留下一个边长为 c 的小正方形。
方法B（面积方程）
同一个大正方形面积有两种表达：
(a+b)² = 4*(ab/2) + c²
即 a² + 2ab + b² = 2ab + c²，约去 2ab 得 a² + b² = c²。
